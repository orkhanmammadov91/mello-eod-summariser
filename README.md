# Mello House EOD Summariser

Zero-configuration weekly report summariser for Mello House.

## How It Works

1. **Drop XLSX files** — Upload 5-7 daily EOD reports
2. **Generate summary** — AI synthesises into prose narratives
3. **Export** — Print to PDF or copy as text

## How to Use

Just open the link in your browser and drop your files!

No setup needed — no URLs, no API keys, no passwords.
